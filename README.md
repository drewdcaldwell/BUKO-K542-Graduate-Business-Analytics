# BUKO-K542-Graduate-Business-Analytics
Improvements in technology to analyze large amounts of data have led businesses to rely heavily on data-driven decision making. In this course students will learn to use both descriptive, predictive, and prescriptive analytics to inform business decisions. Utilize dashboards, regression analysis, forecasting, and optimization to evaluate data.
